##### IDEA 开发个人所喜爱的几款插件（在此推荐）

~~~
  1. 插件名称：lombok
     插件用途：自动生成Getter&&Setter方法，以及自动生成有参无参构造函数，builder建造者模式，toString等等方法
     插件优点：节约开发建造实体时间，让机械的事情通过简单的注解可以实现，也是以后学习途中能写出真正利于开发代码的一个标杆
     插件缺点：针对单个属性字段处理没有那么方便，或者是因为自己使用经验不足
     推荐指数：☆☆☆☆

  2. 插件名称：Alibaba Java Coding Guidelines
     插件用途：规范程序员编码习惯，纠正从入行以来不经意间遗留的代码恶习
     插件优点：对以后编码生涯有着难以言喻的好处
     插件缺点：暂未能想到
     推荐指数：☆☆☆☆☆
     
  3. 插件名称：GsonFormat
     插件用途：处理json字符生成实体类
     插件优点：除却一字一码的手巧，竟然还能有这么方便的插件帮助我们json转实体，实在意外
     插件缺点：暂未能想到
     推荐指数：☆☆☆☆☆
~~~
